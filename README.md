
# ImageSearchApp

An Image Loading app from Unsplash API to demonstrate the application of Advanced Android concepts of Dependency Injection using Dagger Hilt , Paging Library and Kotlin

# Used Libraries
- Navigation Component
- viewBinding
- Dagger Hilt
- Retrofit
- GSON
- Glide
- Paging3
